Теория
1.Переменные
Переменные в Python объявляются простым образом, прописав в терминале x = 0, мы создадим переменную равную нулю. Переменной x можно обозначить разные типы данных, такие как:int, float, str, bool
Пример:x = 1
str(x)
print(x)
Вывод:x выведется в виде 1 с типом данных строки.
2.Ввод-вывод данных
Ввод и вывод данных в Python осущестляется при помощи 2 функций: print(), и input().
Если внутрь скобок в функции print() записать любые символы, будто цифры, буквы, то он выведет содержимое в терминал
!Важно:Буквы нужно отделять кавычками, либо ординарными,либо двойными, одновременное использование разных кавычек приведет к ошибке. Пример:print(123,'adb')
Вывод:123 adb
